## Inizia

Dopo aver clonato la repo tramite git, inizia il server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Apri [http://localhost:3000](http://localhost:3000) con il browser per vedere il risultato.
